HTML

HTML (HyperText Markup Language) n'est pas un langage de programmation : c'est un langage de balisage qui sert à indiquer au navigateur comment structurer les pages web visitées. Le HTML se compose d'une série d'éléments avec lesquels vous pouvez encadrer, envelopper ou baliser différentes parties du contenu pour les faire apparaître ou agir d'une certaine manière. Les balises sont utilisées pour indiquer au navigateur la structure du site. C'est-à-dire que lors de l'écriture de code en HTML, les balises seront interprétées par le navigateur, produisant ainsi la structure et le contenu visuel de la page.

Structure de base d'un document HTML

Un document HTML reçoit des balises qui forment sa structure de base

Paragraphe <p>

Si nous voulons que une ligne dans le text reste en l'état, nous pouvons dire qu'il s'agit d'un paragraphe en l'enveloppant d'un élément paragraphe <p>.

Exemple:

<p>Mon chat est très grincheux </p>

Italique <em>

La balise <em> doit mettre en relief la ligne en l'écrivant en italiques.

Exemple:

<em>Mon chat est très grincheux </em>>

Grass <strong>

Sert à marquer un texte sur lequel on veut insister, il sera généralement affiché en gras. Est une balise imbriquée. <strong>.

Exemple:

<p>Mon chien est très <strong>gentil </strong><p>

Element <a>

Il représente une ancre et permet de transformer en lien l'élément qu'il enveloppe. Il peut recevoir un certain nombre d'attributs

Exemple:

<p>Mon chat est très grincheux </p>
