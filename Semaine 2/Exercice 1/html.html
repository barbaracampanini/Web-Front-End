HTML

HTML (HyperText Markup Language) n'est pas un langage de programmation : c'est un langage de balisage qui sert à indiquer au navigateur comment structurer les pages web visitées. Le HTML se compose d'une série d'éléments avec lesquels vous pouvez encadrer, envelopper ou baliser différentes parties du contenu pour les faire apparaître ou agir d'une certaine manière.

Liste de balises importantes

Voici une liste de balises importantes pour la construction de web sites en utilisant HTML

Paragraphe <p>

Si nous voulons que une ligne dans le text reste en l'état, nous pouvons dire qu'il s'agit d'un paragraphe en l'enveloppant d'un élément paragraphe <p>.

Exemple:

<p>Mon chat est très grincheux </p>
